# **App Name**: Blak Whyte Studio Hub

## Core Features:

- Service Showcase (Public): Display a categorized list of all services with descriptions, prices, durations, and images, pulled from the Firestore database for public viewing.
- Online Booking System (Public): Enable clients to book appointments through an intuitive, multi-step process (service selection, date/time selection, and personal details). Store bookings in the Firestore database.
- Admin Dashboard (Admin): Provide a secure admin panel for managing bookings, services, products, and viewing key statistics (revenue, bookings) pulled from Firestore. Enable actions for each booking (Confirm/Decline, update payment status, initiate WhatsApp chat).
- Automated Invoice Generation (Admin): Automatically generate professional invoices for bookings based on data in Firestore, available to download or print.
- Business Trend Analyzer (Admin): Use booking data from Firestore as a tool to suggest pricing adjustments by identifying popular services and revenue trends over time using generative AI, which helps optimize service offerings and profitability.
- Image Gallery (Public): Showcase a responsive masonry-style grid of images that capture the salon's finest work.
- Product Display (Public): Showcase all physical products sold by the studio, with an image, name, description and price.

## Style Guidelines:

- Primary color: Light gold (#D4AF37) to convey luxury and sophistication.
- Background color: Desaturated gold (#F0EAD7), providing a neutral, elegant backdrop that complements the primary color and accentuates the content. Chosen for light mode.
- Accent color: Soft gray (#8E816A) for a subtle highlight, allowing important elements to stand out without overpowering the design.
- Font choice: 'Inter', a sans-serif typeface, is selected for its clean, readable design that maintains a modern aesthetic and supports comfortable reading on both desktop and mobile platforms.
- Line-based icons using `lucide-react` for a simple, elegant aesthetic.
- A clean, grid-based layout with ample whitespace for a high-end feel.
- Subtle animations and smooth transitions to enhance user experience.
